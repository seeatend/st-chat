### Getting Started

Setup:

```
git clone https://github.com/betop/st-chat.git
```

Start:

```
cd st-chat
meteor
Go to localhost:3000
You can create a new account on localhost like the following:
Username: email@company.com
Password: password
Confirm your email account.
```
